# ⚡ Смета электрика

Приложение для быстрого расчёта сметы на электромонтажные работы. Работает как обычное приложение на Android (PWA) — без Play Store, устанавливается по ссылке и работает офлайн.

**Установить:** [tores-kun.github.io/smeta](https://tores-kun.github.io/smeta/)

## Что умеет

- Прайс-лист на 73 позиции (8 разделов): подготовительные работы, электропроводка, электроустановочные изделия, щитовое оборудование, освещение, тёплый пол, слаботочка, подключение оборудования
- Выбор работ и ввод количества — сумма считается автоматически
- Несколько смет одновременно — для разных клиентов и объектов
- Быстрый ввод: комбо-позиции вроде «розетка на N точек» сами проставляют нужные строки (отверстие + коробка + монтаж)
- Настройки: можно менять расценки, добавлять свои позиции, скрывать то, что редко нужно
- Печать / экспорт в PDF (через диалог печати браузера) и в Excel
- Всё хранится локально на телефоне, интернет не нужен (кроме первой загрузки и экспорта в Excel)

## Как установить на телефон

1. Откройте [tores-kun.github.io/smeta](https://tores-kun.github.io/smeta/) в Chrome на Android
2. Меню (три точки) → **«Добавить на главный экран»** / **«Установить приложение»**
3. Готово — иконка на рабочем столе, работает как обычное приложение

## Для разработки

Чистый HTML/CSS/JS, без сборки и зависимостей. Чтобы обновить прайс или логику — правьте `data.js` / `app.js` / `combos.js` и коммитьте в `main`, GitHub Pages подхватит изменения сам. Не забудьте поднять номер версии в `version.js` — так у пользователей появится плашка «Доступно обновление».

| Файл | Назначение |
|---|---|
| `index.html`, `style.css` | разметка и стили |
| `data.js` | прайс-лист по умолчанию |
| `app.js` | сметы, расценки, экспорт |
| `combos.js`, `combo-ui.js` | комбо-позиции |
| `sw.js`, `manifest.json` | офлайн-режим и установка как приложение |
