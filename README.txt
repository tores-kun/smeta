СМЕТА ЭЛЕКТРИКА — приложение для расчёта сметы
================================================

Что это
-------
Веб-приложение (PWA), которое устанавливается на Android как обычное
приложение и работает офлайн. Прайс-лист взят из файла
"смета электрика.xlsx" (8 разделов, 73 позиции).

Как установить на телефон
--------------------------
1. Разместите папку "smeta-app" там, где к ней есть доступ с телефона
   (например, в облаке — Google Диск/Яндекс.Диск — или на веб-хостинге).
   Проще всего: загрузите папку на любой бесплатный статический хостинг
   (netlify.com — просто перетащить папку на сайт, получится ссылка)
   либо откройте index.html напрямую в Chrome на телефоне через файловый
   менеджер.
2. Откройте ссылку (или файл index.html) в Chrome на Android.
3. Нажмите на меню (три точки) → "Добавить на главный экран" /
   "Установить приложение".
4. Готово — иконка появится на рабочем столе, приложение открывается
   как обычное, работает без интернета.

Возможности
-----------
• Выбор работ из прайса по разделам, поиск по названию
• Ввод количества — сумма считается автоматически
• Несколько смет (меню ☰) — для разных клиентов/объектов
• Редактирование расценок и добавление своих позиций (кнопка ✎)
• Печать / сохранение в PDF (через диалог печати Chrome)
• Экспорт в Excel (.xlsx)

Все данные (сметы, изменённые расценки) хранятся локально на телефоне.
Для экспорта в Excel один раз нужен интернет (подгружается библиотека),
дальше приложение работает офлайн.
